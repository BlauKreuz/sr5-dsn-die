# Option for customizing Dice-So-Nice die for Shadowrun 5.0",
  "description": "A simple module to disable the forced SR5 custom die (ds) in Dice-So-Nice so that the dot versions can be used.
It is also possible to customize the dot versions by replacing the dot.webp image in \modules\dice-so-nice\textures folder with one's own design. A sample image is available.
This module does not do anything if Dice-So-Nice module is not installed and in use.

Scene reload is needed for the changes to take effect.
